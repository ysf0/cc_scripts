You can kill any process by given port number:

~ c_port_kill_all_process "8089"         
    
  ** executing command:  lsof -i:8089 
  COMMAND   PID    USER   FD   TYPE    DEVICE      SIZE/OFF   NODE NAME
  firefox   23621  jack   3u   IPv4    231990      0t0        TCP *:8089 (LISTEN)
  ** executing command:  kill -9 23623



Open a simple server-socket which serves a simple text:

~ c_port_open_serving_string "hello" 8089
  
  ** executing command:  eval printf %s hello | nc -n -v -l 8089 
  Listening on 0.0.0.0 8089



If you don't know the parameters, the function will not work and it will warn you. Or you get get help:

~ c_port_open_serving_string hello

  ** check the parameters. use -h to see the parameters.

~ c_port_open_serving_string -h

  *** required parameters
  ** TEXT_TO_SHARE
  ** LOCAL_PORT
